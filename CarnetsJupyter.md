Les Carnets de Jupiter (ou Jupyter Notebooks)

1. C'est quoi ? (Définition simple)
   Un carnet Jupyter est un document interactif (fichier .ipynb) qui permet de mélanger trois choses au même endroit :

Du texte riche : Pour expliquer ta démarche (titres, listes, images).

Du code informatique : Que l'on peut exécuter directement (Python, R, Julia...).

Des résultats : Les graphiques et calculs s'affichent instantanément sous le code.

Imagine : C'est comme une page Wikipedia où tu aurais le droit d'écrire du code et de voir les graphiques se dessiner en temps réel.

2. Qui l'a créé et pourquoi ? (L'Histoire)
   L'inventeur : Fernando Pérez, un physicien colombien, en 2001.

Le but : Il voulait un outil pour la "Science Ouverte". À l'époque, les logiciels de calcul étaient chers et fermés. Il voulait que n'importe quel chercheur puisse partager ses calculs gratuitement pour que d'autres puissent les vérifier (c'est ce qu'on appelle la reproductibilité).

L'anecdote du nom : "Jupyter" vient des langages Julia, Python et R. C’est aussi un clin d’œil aux carnets de notes de Galilée qui observait la planète Jupiter.

3. À quoi ça sert ?
   Analyse de données : Nettoyer des chiffres et créer des graphiques.

Enseignement : Les profs créent des cours interactifs où l'élève peut modifier le code pour comprendre.

Intelligence Artificielle : C’est l’outil n°1 pour entraîner des IA aujourd'hui.

4. Comment l'utiliser ?
   Le document fonctionne par "cellules" (des blocs) :

Cellule Markdown : Pour écrire ton texte et faire la mise en page.

Cellule de Code : Pour taper ton programme.

Bouton "Exécuter" (ou Maj + Entrée) : Pour lancer le bloc de code et voir le résultat.

On peut l'utiliser directement dans VS Code (en installant l'extension "Jupyter") ou sur internet via Google Colab.

5. Est-ce que c'est gratuit ?
   Oui, 100 % gratuit. C’est un logiciel Open Source (libre). Personne ne possède Jupyter, c’est une communauté mondiale qui l'entretient pour que tout le monde puisse coder sans payer.
