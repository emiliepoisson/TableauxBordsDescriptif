# TableauxBordsDescriptif

C'est quoi Les Carnets de Jupyter ?

Un document interactif qui mélange texte, code et résultats au même endroit.
Tout commence en 2001. Fernando Pérez, un physicien colombien alors doctorant à l'Université du Colorado, est frustré. Pour ses recherches, il doit jongler entre plusieurs logiciels propriétaires (et chers) pour analyser ses données.

L'Histoire des Carnets de Jupyter
Créateur : Fernando Pérez (2001).

But : Créer un outil gratuit pour la science ouverte afin que les chercheurs puissent partager leurs preuves et leurs calculs facilement.

Nom : Un mélange de Julia, Python et R. C'est aussi un hommage aux carnets de notes de l'astronome Galilée sur les lunes de Jupiter.
Les Carnets Jupyter :
un environnement dans lequel il est possible de combiner le code informatique et le texte en langage naturel.

Pourquoi c'est génial ?
Cellules : On exécute le code petit bout par petit bout (pas besoin de tout relancer).

Visuel : Les graphiques s'affichent directement sous le code.

Partage : GitHub affiche le rendu des carnets automatiquement, ce qui facilite le travail d'équipe.
